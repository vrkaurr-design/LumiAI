 # LumiAI
 
 A Next.js (App Router) frontend for virtual try-on, AI skin analysis, and a small e-commerce flow (shop + cart + auth).
 
 ## Brief
 
 - **Virtual experiences**
   - Virtual try-on and AI skin analysis flows
 - **Commerce**
   - Product catalog with filtering/search and product cards
   - Cart with persisted items, quantity controls, and totals
 - **UI**
   - Tailwind-based UI with animated/live background
   - Sign-in / sign-up pages
 
 ## Tech Stack
 
 - Next.js (App Router) + TypeScript
 - Tailwind CSS
 - Zustand (cart persistence)
 
 ## Run Locally
 
 ```bash
 cd frontend
 npm install
 npm run dev
 ```
 
 Then open `http://localhost:3000`.

 **Requirements**

 - Node.js `>= 18.17`
 - npm (or your preferred package manager)

 **Windows notes**

 - Use the same commands in PowerShell/CMD (no `npm.cmd` needed)
 - If you see script execution policy issues, try running in CMD or enable scripts for your shell
 
 ## Common Scripts
