# API Changelog

## v1.0.0 (Initial Release)

### Features
- **Authentication**: JWT-based Signup/Login (`/api/v1/auth`)
- **Products**: Catalog management, filtering, and search (`/api/v1/products`)
- **Skin Analysis**: AI-powered image analysis for skin tone/type detection (`/api/v1/analysis`)
- **Orders**: Cart management and checkout flow (`/api/v1/orders`)
- **Admin Dashboard**: Analytics and inventory management (`/api/v1/admin`)
- **Payment**: Mock Stripe integration (`/api/v1/payment`)
- **Security**: Rate limiting, Helmet headers, CORS.
